# Sathvik Poreddy — Portfolio

Premium AI-centered developer portfolio built with Next.js, Tailwind CSS, and Framer Motion. It highlights AI impact, research, and scalable engineering.

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion

## Scripts
- `npm run dev` — launch development server
- `npm run build` — compile for production
- `npm run start` — run production server
- `npm run lint` — run Next.js linting

## Deployment
- Static site generation with `next build`
- Compatible with Netlify (ensure `npm run build` passes)
- Includes modern hero, project, research, and experience sections with animation

## Notes
- Edit content in `src/app/page.tsx`
- Tailwind configuration lives in `tailwind.config.js`
- Global styles under `src/styles/globals.css`
